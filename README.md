# pandas-# Titanic Data Analysis using Pandas

This project demonstrates **data preprocessing, cleaning, and visualization** using the Titanic dataset with Python libraries such as **Pandas, NumPy, Seaborn, and Matplotlib**.

The goal of this project is to practice **Exploratory Data Analysis (EDA)** and understand relationships between different variables in the dataset.

---

## Project Overview

The Titanic dataset contains passenger information such as:

- Passenger class
- Gender
- Age
- Fare
- Embarkation port
- Survival status

This project focuses on:

- Handling missing values
- Cleaning the dataset
- Feature engineering
- Correlation analysis
- Data visualization

---

## Technologies Used

- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- Google Colab / Jupyter Notebook
